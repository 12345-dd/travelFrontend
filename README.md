# AI-Travel Planner — Frontend

An AI-powered travel planning application that helps users generate personalized travel itineraries, hotel recommendations, budgets, and packing lists using Google Gemini AI.

**Live Demo:** [https://smarttripai.netlify.app](https://smarttripai.netlify.app)  
**Backend Repo:** [https://github.com/12345-dd/travelBackend](https://github.com/12345-dd/travelBackend)

---

## Features

- **JWT-based Authentication** — Secure signup & login
- **AI Trip Generation** — Personalized travel itineraries
- **Travel Dashboard** — Manage all your trips in one place
- **Hotel Recommendations** — AI-suggested hotels for every destination
- **Budget Planner** — Estimated travel budget breakdown
- **Packing Checklist** — AI-generated packing suggestions
- **Day Regeneration** — Regenerate a specific day using AI
- **Activity Management** — Add or remove itinerary activities
- **Protected Routes** — Only authenticated users can access their data

---

## Getting Started (Local Setup)


### 1. Clone the Repository
```bash
git clone https://github.com/12345-dd/travelFrontend.git
cd travelFrontend
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Run the App
```bash
npm run dev
```

> App runs on `http://localhost:5173`
